# Noah Neri — Portfolio

> Personal portfolio site. Dark, technical, custom-built. Purpose: land employment by showcasing the ability to use AI tools to ship real projects fast.

**Status:** Live on Vercel | Build verified | Synced to GitHub `main`

- **Live site:** [https://noahneriportfolio.vercel.app/](https://noahneriportfolio.vercel.app/)
- **Who it's for:** employers, collaborators, and anyone evaluating Noah as a builder
- **What it does:** showcases shipped work, resume, contact info, and project case studies

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
